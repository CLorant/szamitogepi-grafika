## 1. gyakorlati feladat: Téglatestek

### Projekt leírása
Ez egy egyszerű program egy felhasználó által megadott téglatesten való számítások végzésére.

### Főbb funkciók
- Térfogat számítás
- Felület számítás
- Van-e négyzetes lapja a téglatestnek

### Fordítás és Futtatás
```bash
make && cuboid.exe
```
